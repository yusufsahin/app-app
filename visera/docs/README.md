# @visera Documentation

## Scope
This section defines `@visera` as an enterprise-ready diagram platform that can run in three modes:

- Embedded Mode (inside ALM)
- Standalone Mode (independent product)
- Headless Mode (API/SDK consumption)

## Product Intent
`@visera` is not a single drawing canvas. It is a platform for structured diagram creation, governance, integrations, and AI-assisted workflows.

## Delivery Priority
Core first. Advanced integrations, AI expansion, and custom canvas tracks follow only after core completion criteria are met.

## ALM Dual Model
`@visera` supports a dual product strategy:

- Embedded in ALM for low-friction adoption
- Sold as a standalone product with its own tenant/workspace model

The core architecture remains product-agnostic so the same domain and platform modules work in both models.
