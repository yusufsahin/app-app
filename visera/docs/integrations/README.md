# Integration Hub

## Scope
Integration Hub connects `@visera` with design tools, ALM systems, SCM platforms, and AI provider ecosystems using consistent contracts.

## Goals
- Standard connector architecture
- Secure credential handling
- Auditable integration actions
- Reusable integration lifecycle

## Integration Domains
- Design tools: Figma and Figma Make
- ALM systems: Jira and Azure DevOps
- SCM systems: GitHub and GitLab
- AI providers: public/private/in-house endpoints
- Eventing: inbound and outbound webhooks
